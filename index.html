    Three essential technologies for the WWW

    1. URL (Uniform Resource Locator) = Address of a given unique resource on the Web.
    2. HTML = publishing language
    3. HTTP = Hytertext Transfer Protocol = Protocol for fetching resources such as HTML documents.

    URL

    A Uniform Resource Locator describes the specific location of a resource on a particular server.
    It tells you exactly how to fetch a resource from a precise, fixed location.
    A URL is a specific type of URI (Uniform Resource Identifier).
    Most commonly URLs refferences to web pages (HTTP), but are also used for file transfer (FTP), email (mailto), database access (JDBC) etc.

    A URL is composed from an allowed set of ASCII characters consisting of reserved characters and unreserved characters and the character %.

    A typical URL contains
    1. A scheme that identifies the protocol to be used to access the ressource on the Internet, it can be HTTP or HTTPS.
    2. A host The hostname identifies the host of the roesource (e. g. www.example.com).
    3. A path
    4. A query string, it follows the path and provides a string of information.

    Web Server

    A web server is computer software and underlying hardware that accepts requests via HTTP.
    A web browser or web crawler makes a request for a ressource (e. g. web page) using HTTP.
    The server responds with the content of the resorce or an error message. (describe)
    A web server can also accept and store resources sent from the user agent.

    Embedded systems are the smallest web servers, e. g. a router that runs one as its configuration interface.
    A high-traffic website might handle request with hundreds of servers.
    A resource sent from a web server can be a preexisting file (static content) or it can be generated by the time of request (dynamic content) by another program.

    Web Client (Web Browser; User Agent)

    A web browser accesses the World Wide Web or a local website. The web browser retrieves the requested web page files from a web server and then graphically renders the page on the user's screen.

    A search engine is not a Web Browser! It provides links to other websites.

    HTML

    All HTML documents must start with a !DOCTYPE declaration; it tells the browser what document to expect.

    The body element contains all the contents of an HTML document.
    The head element is a container for metadata (data about data).
    Metadata is about the HTML document; it's about the style, character set, scripts, etc.
    The em element is for emphasized text; f.e.: example
    The html tag is the container for all other HTML elements (except for !DOCTYPE tag).
    The strong tag is to define important text in a document. It is displayed in bold.
    The title tag defines the title of the document. It is required in HTML documents.

    HTML

    2.1. HTML Basic Tags

    All HTML documents must start with a <!DOCTYPE html> declaration; it tells the browser what document to expect.

    The article tag is for independent, self-contained content
    The html tag is the container for all other HTML elements (except for !DOCTYPE tag).
    The body element contains the visible part of an HTML document.
    The em element is for emphasized text; f.e.: example
    The section tag defines a section in a document The strong tag is to define important text in a document. It is displayed in bold.

    The a tag is for links, e.g.: <a> href="https://www.w3schools.com">This is a link</a>

    2.2. HTML Knowledge about Elements

    A HTML element contains of a start tag, the element itself and an end tag.
    There are also void HTML Elements = doesn't contain any content; e.g.: <br>; it defines a line break
    HTML is not Case Sensitive, you can mark a paragraph with P and with p

    2.3. HTML Attributes

    HTML attributes provide additional information about HTML elements
    The href attribute specifies the URL of the page the link (with a tag) goes to.
    The style attribute: to add styles
    The lang attribute (of the html tag): declares language of the Web page
    The title attribute defines extra information about an element
    e.g.: <p title="John 'ShotGun' Nelson">

    2.4. HTML Headings and Paragraphs

    The h1 tag defines a heading, headings are h1 to h6 tags.
    Note: I will continue to use the br Tag because it is more comfortable to read without the white spaces.
    The p tag defines a paragraph, browsers add some white space before and after it.
    The hr tag defines a horizontal line between the content
    The br tag defines a line break.

    The pre tag defines a preformatted text. e.g.: this text with blue in style

    2.5. HTML Comments and Links

    The comment tag: e.g.: <!-- text as a comment -->
    The a tag defines a hyperlink; e.g.: <a> href="https://www.google.com">Visit google!</a>
    The target attribute can be used so that the link won't open on the current browser window.
    It can have one of these values:
    _self: default; opens it in same window/tab
    _blank: opens it in new winow / tab
    _parent: opens it in parent frame
    _top: opens it in the full body of the window
    There is an absolute URL and a relative URL (relative: does not include domain name)
    The 'mailto:' tag: to open a link to open a mail adress with a mail program
    The button tag: to use an HTML button as link (has to be with JavaScript code!)
    The title tag: extra information, shows when the mouse moves over it.

    2.6. HTML Images

    The img tag is for images on a HTML page; the src attribute specifies the path
    If the image is in another folder, the folder name must be included in the src.
    If the image is on another server / website, the absolute URl must be included in the src.
    e.g.: <img src="img_girl.jpg">
    The alt attribute for the image contains the alternate text for an image if it can't get displayed.
    The width and height attributes: e.g.: <src="img_girl.jpg" width="500" height="600">
    Image as Link: just put the img tag inside the a tag.

    2.7. HTML Head Element

    The head element is a container for metadata (data about data).
    Metadata is about the HTML document; it's about the style, character set, scripts, etc.
    The title tag defines the title of the document. It is required in HTML documents.
    1) it defines a title in the toolbar
    2) it provides a title for the favorits
    3) title of the search enginge-results
    The meta tag specifies the character set, description, keywords, author of the web and viewport settings.
    It won't be displayed on the page, but is used by browsers, by search engines and other web services.
    <head>
    <meta charset="UTF-8">
    <title>2. Cheat Sheet</title>
    </head>

    CSS

    3.1. CSS Syntax, Selectors and Comments


    selector: points to the HTML element to style
    declaration: declarations separated by ;
    element selector: selects Html elements based on the element name
    grouping selector: groups selectors to minimize code; e.g.: h1, h2, p { style}
    comments: with /* and */

    3.2. CSS Color and Background

    color Names: there are perdefined color names in CSS e.g.:
    <h1 style="color:Tomato">test</h1>
    Hexadecimal: #rrggbb (red, etc. in HEX; 00 and ff (0-255)) e.g.:
    <h1 style="background-color:#ff6347;">test</h1>
    background: e.g.:background-color: lightblue = set background color of an element
    image as background: e.g.: background.image: url("paper.gif");
    repeat background: horizontally: background-repeat: repeat-x;
    vertically: -y; no repeat: background-repeat: no-repeat;
    background position: e.g.: background-position: right top;

    3.3. CSS Text and Fonts

    text color: e.g.: color: blue;
    text allign: e.g.: text-align: center;
    text decoration: e.g.: text-decoration-line: underline; (-color, -style, -thickness)
    shorthand property: text-decoration; e.g.: text-decoration: underline red double;
    text transformation: capitalization of text; e.g.: text-transform: uppercase;
    Serif: small stroke at edges of letter: for formality and elegance
    Sans-serif: without it, clean lines: for modern and minimalistic look
    font-family: font of text and if it is not available, let browser pick similar from same family; e.g.: font-family:'Courier New', Courier, Monospace
    font-style: e.g.: font-style: italic;
    font-size: e.g.: font-size: 40px; (with pixels) font-size: 2.5 em; (with em, to adjust size in all browsers)
    font-weight: how thin / thick characters are, e.g.: font-weight: bold;

    3.4. CSS Link and Combinators

    Link Color: style links; e.g.: a {color: hotpink;}; four link states:
    a:link - normal, univisited; a:visited; a:hover - when mouse moves over; a:active: moment when clicked
    All CSS Combinator Selectors:
    1. element element; e.g.: div p; selects all p elements inside div elements = descendant selector
    2. element>element; e.g.: div > p; selects all p element where the (direct) parent is a div element = child selector
    3. element+element; e.g.: div + p; selects the first p element that are placed immediately after div elements = adjacent sibling selector
    4. element~element; p ~ ul: selects every ul element that are preceded by a p element = general sibling selector
    e.g.: div p { background-color: yellow; }

    3.5. CSS Pseudo-classes

    Pseudo classes: define a special state of an element;(=style element under certain conditions)
    e.g.: style an element when the mouse moves over; Syntax:
    selector:pseudo-class {
    property: value;
    }
    Anchor Pseudo-classes: links can be displayed in different ways: e.g.:
    a: link { /* univisited link */
    color: #FF000;
    }
    Hover on div: changes color of element when hovers - example:
    div:hover {
    background-color: blue;
    }
    :first-child: it matches a specified element that is the first child of another element

    4.3. HTML for Tables

    defintion:
    <table>
    <tr>
    <th>Country</th>
    <th>Company</th>
    <tr>
    <td>Germany</td>
    <td>Volkswagen</td>
    </tr>
    </table>

    CountryCompany
    GermanyVolkswagen

    td: defines a cell
    th: defines a header cell
    tr: defines a row
    thead: groups head content tbody: groups body content
    tfoot; groups footer content
    colspan: attribute that defines the number of columns a cell should span
    rowspan: to make a cell span over multiple rows
    caption: tag to insert a caption to a table

    4.4. CSS for Tables

    border: to specify table borders
    border-collapse: to set if the table borders should collapse into a single border or be seperated (standard)
    width: 100%: to set width of table
    height: to set height of table
    border: 1 px solid: for just one border around the table
    border-style: property for border, e.g.: dotted
    border-width: e.g.: solid
    border-color: e.g.: red
    border-right-style: dotted: individual slide; right border is dotted
    shorthand property: border is a shorthand property for border-width, border-color and border-style
    rounded borders: e.g. border-radius: 5 px;
